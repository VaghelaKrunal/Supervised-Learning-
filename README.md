## 📌 Project Overview

This project was developed as part of a **Predictive Insight Engine – Supervised Learning** assignment.

The objective is to build an end-to-end Machine Learning pipeline capable of predicting house prices using multiple regression techniques. The project demonstrates every major stage of a supervised learning workflow, including data preprocessing, exploratory data analysis (EDA), model development, optimization, evaluation, and business interpretation.

Rather than focusing on only one algorithm, this project compares multiple regression models to understand their predictive performance, assumptions, advantages, and limitations.
